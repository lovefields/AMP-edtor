    에디터 사용법

    • 기본적으로 포스트는 AMP(모바일 최적화 페이지) 라서 코드상의 문제가 없어야합니다.
      검수는 AMP Test 에서하며 문제가 생길시 관리자에게 헬프를 요청하세요.
    • 기본적인 효과/추가는 왼쪽 컨트롤창 에서 가능합니다.
    • 속성에 관한 것은 오른쪽 컨트롤창 에서 가능합니다.
    • 폰트 컬러,사이즈,이미지 사이즈 의 추가는 관리자에게 문의하세요.
    • 되도록 에디터 내에서 복사는 조심히 사용하셔야 합니다.
      단락복사 X / 문장 복사 O
    • 오른쪽 컨트롤창 에서 가끔 모든 속성이 나오는것은 정상입니다.
    • 되도록 테이블의 col은 4개 이하가 모바일에서 좋습니다.
    • 여러줄 드래그 속성 변경도 불가능합니다.
    • 이미지는 가로 3000px이하로 사용하시기 바랍니다.
    • 일러스트 떄문에 한장당 2mb로 제한중입니다. 되도록 일러스트를 제외한 이미지는 가로 1200px 용량 1mb 미만으로 사용하시기 바랍니다.
